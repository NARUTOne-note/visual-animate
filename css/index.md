# css 动画/可视化

- **Transfrom 变形**：Transfrom 用于改变元素的位置、大小、旋转和斜切等属性。用于创建元素的动态效果，包括移动、淡入淡出、旋转等。
- **Animation 动画**：用于创建元素的动态效果，包括移动、淡入淡出、旋转等。
- **Filter 过滤器**：用于对元素应用各种效果，如模糊、颜色调整、阴影等。
- **Gradient 渐变**：用于创建平滑过渡的颜色效果，可以是线性渐变或径向渐变。
- **Mask 遮罩**：mask 和 mask-image 中的遮罩图像中的不透明部分将显示出元素的内容，透明部分将隐藏元素的内容。
- **Function 函数**：用于在 CSS 中执行各种数学运算、颜色转换和其他复杂的计算操作。
- **Houdini**：CSS Houdini 是一组用于扩展和自定义 CSS 的 API 集合，我们可以使用 Paint Worklet 自定义元素绘制、Layout Worklet 自定义布局、Animation Worklet 自定义动画。

## 合理使用 Transform

transform 拥有自身独立上下文是渲染优化的关键，它可以轻易地与原视图组合重绘而不用花费主线程大量计算时间进行重排。

使用 transform 属性时，浏览器会创建一个`新的渲染层（也称为合成层或者 GPU 层）`。这个渲染层是在页面的元素上进行变换操作时创建的，它在浏览器窗口的层次结构中处于独立的层级。

这个新的渲染层可以提供更高效的渲染和动画效果，因为它可以利用硬件加速来进行图形计算和渲染。创建新的渲染层也会带来一些性能开销。过多地创建渲染层可能会消耗更多的内存和处理资源，从而影响页面的性能。因此，在使用 transform 属性时，需要谨慎使用，并避免不必要的渲染层的创建。

如果只涉及到简单的平面变换，如旋转、缩放和平移等，2D 变换通常会更高效。而如果需要实现更复杂的三维效果或者涉及到透视变换等情况，3D 变换可能更适合.
